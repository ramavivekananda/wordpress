Create shortcode for dropdown text or scrollable text

== Description ==

create shortcode for dropdown text or scrollable text.<br>
Create Dropdown text : <br>[dropdowntext]TEXT HERE[/dropdowntext]<br>
Create Scrollable text : <br>[scrollabletext]TEXT HERE[/scrollabletext]<br>
set dropdown text height in shortcode : <br>[dropdowntext height=200]<br>
set theme and height of scrollable text : <br>[scrollabletext theme="dark" height=200]<br>

== Installation ==

This plugin can be installed directly from your site.
<br>
Log in and navigate to Plugins <br> Add New.<br>
Type Dropdown and scrollable Text into the Search and hit Enter.<br>
Locate the Dropdown and scrollable Text plugin in the list of search results and click Install Now.<br>
Once installed, click the Activate link.<br>

It can also be installed manually.
<br>
Download the Dropdown and scrollable Text plugin from WordPress.org.<br>
Unzip the package and move to your plugins directory.<br>
Log into WordPress and navigate to the Plugins screen.<br>
Once installed, click the Activate link.
